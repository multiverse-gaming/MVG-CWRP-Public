# CWRP
CWRP is a roleplaying game built on s&box with a session-based gameplay. In this game, players roleplay as roles within the Grand Army of the Republic, engaging in battle against adversaries and conducting trainings, each session creates an unique roleplaying experience.

This repository is the public counterpart to the CWRP repository and includes the SDK, Server Image, and issue/project tracking.

# Features
* Combat System: Incorporated simple weapon base with Star Wars inspired blasters.
* Job Menu : Select from a range of custom job's with unique loadout's to roleplay as.
* Roleplay Gameplay: Mimics the core mechanics of DarkRP while providing a unique, session-only experience.
* Admin Interface: An in-game command menu to moderate the server.

# Gameplay
* Players take on different roles and interact with the map and players around them.
* Shooter gameplay using weapons to partake in player versus player and player versus enemy combat.
* Session-based gameplay ensures unique experiences each time a player joins.
